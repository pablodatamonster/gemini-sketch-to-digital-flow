# Nano Banana Design System

This repository contains structured instructions for Gemini (Nano Banana 2) to automate the transition from hand-drawn sketches to digital assets.

## Workflow Files
1. **01-wireframe-specs.md**: Structural layouts with light grey outlines.
2. **02-mockup-specs.md**: Clean, black-ink digital versions of sketches.
3. **03-prototype-specs.md**: High-fidelity, branded dashboard designs.

## How to Use
1. Start a new chat with Gemini.
2. Upload the relevant .md file from this repo.
3. Upload your hand-drawn sketch or wireframe.
4. Gemini will generate the digital asset following these specific constraints.
